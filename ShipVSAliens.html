Ship vs Aliens

Octobe 2018 - January 2019

Ship vs Aliens is a video game project which was developed using Python and Pygame. Pygame was an essential component of the project since it provided movement and interaction between the user and the ship. Throughout the game, the game will keep track of the user's lives, current scores, and their highest score. When the user dies, the game will restart itself and provide another opportunity for the user to redeem themselves.
